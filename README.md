 📚 Library Booking MIU Telegram Bot

Telegram-бот для бронирования мест в библиотеке ММУ с поддержкой пользователей и администраторов.

---

 🚀 Возможности

 👤 Пользователь

* Регистрация при первом запуске
* Бронирование места (дата → время → подтверждение)
* Просмотр своих бронирований
* Отмена бронирования
* Защита от переполнения слотов (лимит мест)

---

 🛠 Администратор

* Доступ к админ-панели
* Просмотр всех бронирований
* Просмотр бронирований на сегодня
* Удаление любых бронирований

---

 🧱 Стек технологий

* Python 3.10+
* aiogram 3.x
* SQLite (aiosqlite)
* FSM (Finite State Machine)
* dotenv

 ⚙️ Установка и запуск

 1. Клонировать репозиторий

```bash
git clone https://github.com/your-repo/mteamproject.git
cd mteamproject
```

---

 2. Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate    Linux / Mac
venv\Scripts\activate       Windows
```

---

 3. Установить зависимости

```bash
pip install -r requirements.txt
```

---

 4. Создать `.env` файл

```env
BOT_TOKEN=your_telegram_bot_token
```

---

 5. Указать админа

В файле `app/config.py`:

```python
ADMINS = [YOUR_TELEGRAM_ID]
SLOT_LIMIT = 5
```

---

 6. Запуск бота

```bash
python bot.py
```

---

 📊 Логика работы

 Бронирование

```
Пользователь
   ↓
Выбор даты
   ↓
Выбор времени (с учётом занятости)
   ↓
Подтверждение
   ↓
Сохранение в БД
```

---

 Ограничение слотов

* Каждый слот имеет лимит (`SLOT_LIMIT`)
* При достижении лимита слот блокируется
* Отображается как:

```
10:00 (5/5)
```

 🧪 Тест-кейсы

 1. Регистрация пользователя

**Шаги:**

1. Запустить бота
2. Нажать `/start`

**Ожидаемый результат:**

* Пользователь добавлен в БД
* Отображается главное меню

---

 2. Создание бронирования

**Шаги:**

1. Нажать "Забронировать место 📚"
2. Выбрать дату
3. Выбрать время
4. Подтвердить

**Ожидаемый результат:**

* Бронирование сохраняется
* Сообщение: "Бронирование успешно создано"

---

 3. Защита от переполнения

**Шаги:**

1. Заполнить слот до лимита
2. Попробовать забронировать ещё

**Ожидаемый результат:**

* Ошибка: "Все места заняты"

---

 4. Просмотр бронирований

**Шаги:**

1. Нажать "Мои бронирования 📖"

**Ожидаемый результат:**

* Список всех бронирований пользователя
* Кнопки отмены

---

 5. Отмена бронирования

**Шаги:**

1. Открыть "Мои бронирования"
2. Нажать ❌

**Ожидаемый результат:**

* Бронь удаляется из БД

---

 6. Админ — просмотр всех бронирований

**Шаги:**

1. Зайти под админом
2. Нажать "Админка ⚙️"
3. Ввести `/all_bookings`

**Ожидаемый результат:**

* Показаны все брони

---

 7. Админ — удаление брони

**Шаги:**

1. Открыть список бронирований
2. Нажать ❌

**Ожидаемый результат:**

* Бронь удаляется
* Список обновляется

---

 8. Админ — бронирования на сегодня

**Шаги:**

1. Ввести `/today_bookings`

**Ожидаемый результат:**

* Показаны брони только за текущий день

---

 🔒 Ограничения

* Один слот = ограниченное количество мест
* Дата хранится в формате `dd.mm.yyyy`
* SQLite используется как локальная БД
